# 關於RestaurantList
1. 查看所有的餐廳
2. 查看餐廳詳細資訊

3. 以餐廳名稱或店家類型來搜尋特定餐廳

# 環境建置與需求
1. Node.js 16.14.0
2. Express": 4.16.4
3. Express-handlebars": 3.0.0
4. Bootstrap 5.1.3

# 安裝步驟
1. 安裝node.js 與 npm
2. 下載本專案
3. 透過終端機進入資料夾，輸入：npm install
4. 完成npm安裝後，繼續輸入：npm run start
5. 若看見此行訊息則代表順利運行，打開瀏覽器進入到以下網址 Listening on http://localhost:3000
6. 若欲暫停使用 ctrl + c
